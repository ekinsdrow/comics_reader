# Comics Reader

Simple app for reading comics on desktop and mobile platforms.

## Supported formats:
1. Cbz
2. Folders with sorted images (png, jpg, jpeg, gif)
3*. In plans support for cbr

## Platforms:
1. Android
2. Ios
3. Desktop
